# Example Apps

These examples support the labs in this repository.

## Available Examples

- `plain-static`: no build step, useful for static hosting basics
- `vite-app`: Vite app with public environment variables
- `react-app`: React app powered by Vite
- `next-app`: Next.js app with a simple App Router page

## Safety

The `.env.example` files contain placeholders only. Do not commit real secrets.
