# Code of Conduct

## Our Pledge

We want `web-deploy-lab` to be a welcoming project for learners and contributors with different backgrounds and experience levels.

We pledge to make participation in this project respectful, constructive, and harassment-free.

## Expected Behavior

- Be kind and patient with learners
- Give feedback that helps people improve
- Assume good intent, and ask for clarification when needed
- Keep discussions focused on the project and learning outcomes
- Respect different tools, platforms, and levels of experience

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Dismissive comments about beginners
- Publishing private information without permission
- Spam or off-topic promotion
- Any behavior that makes participation unsafe or unwelcome

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block contributors who violate this code of conduct.

If you see a problem, please report it through the project's GitHub issue or maintainer contact channel.
