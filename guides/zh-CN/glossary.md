# 故障排查术语表

## 404

服务器找不到请求路径。检查部署根目录、路由 fallback 和资源路径。

## 构建命令

托管平台用于生成部署产物的命令，例如 `npm run build`。

## 构建日志

托管平台输出的安装、构建和部署过程日志。优先寻找第一个有意义的错误。

## 缓存

缓存可能让旧文件在部署后仍然可见。需要检查浏览器、CDN 和平台缓存。

## DNS

域名系统记录，用于把域名指向托管平台。

## 环境变量

源码之外提供的命名配置值。

## 输出目录

构建完成后包含可部署文件的目录。

## 预览部署

在合并生产前，从分支或 Pull Request 创建的部署。

## 冒烟测试

一组快速检查，用来确认部署基本健康。

## SSL/TLS 证书

为域名启用 HTTPS 的证书。
