# 安全处理密钥

前端部署经常需要配置。有些值可以公开，有些必须保持私密。

## 公开值

公开值可以出现在浏览器代码中。例如：

- 公开 API 基础地址
- 分析服务项目 ID
- 功能开关名称

## 密钥

密钥不能出现在浏览器 bundle、公开日志或文档中。例如：

- API token
- 私钥
- 数据库密码
- Webhook 签名密钥

## 规则

- 提交 `.env.example`，不要提交 `.env`。
- 使用 `replace-me` 这类占位值。
- 把 `VITE_` 和 `NEXT_PUBLIC_` 变量视为公开值。
- 任何被提交或打印过的密钥都要立即轮换。
- 生产私密值放在托管平台或密钥管理器中。
